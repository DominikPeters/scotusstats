Oral Argument Transcript Videos

I'm maintaining a YouTube channel that publishes videos of Supreme Court oral argument transcripts. The data is taken from oyez.org. The videos make it easy to see who is the current speaker and which speaking turns are coming up. For some past arguments, the audio of the opinion announcements are provided also.

Technically, the videos are made by rendering the data using HTML/CSS, saving the frames to image files using puppeteer by screenshotting, and encoding the video using ffmpeg. The code generating the videos is open source and available on GitHub.

I started this project to provide alternatives to other videos uploaded to YouTube, which either show a still frame only, or otherwise videos of dogs or cartoons, and these did not seem like an appropriate representation of the Court.

Please consider subscribing and liking videos, since that helps with visibility and discoverability.
